# raw_records.jsonl 表示ツール

LogRep_r が出力した `raw_records.jsonl` を、人間が確認しやすい表形式で表示する簡易ツールです。

## 使い方

1. `AssistantTool/index.html` をブラウザで開きます。
2. `raw_records.jsonl` を画面にドラッグ＆ドロップするか、ファイル選択で読み込みます。
3. 表の上段にあるフィルターで値を絞り込みます。
4. セルをクリックして選択し、`Ctrl+C` または「選択範囲をコピー」でコピーします。
5. `Shift` を押しながら別セルをクリックすると、矩形範囲を選択できます。
6. セルをダブルクリックすると、そのセルの値を即時コピーします。

## 表示仕様

- JSON の key 名を列名として表示します。
- `meta_fields` は `meta1` から `meta21` の固定列に展開します。
- `visible_text` は JSON 読み込み時に Unicode エスケープを通常テキストへ変換します。
- JSONL の解析に失敗した行は、`parse_status=error` として表に残します。
- コピー内容は TSV 形式です。Excel などの表計算ソフトへ貼り付けやすい形式です。

## 補足

外部ライブラリは使用していません。Windows 環境では、`index.html` を Edge などのブラウザで直接開いて利用できます。
